Q 01 নতুন স্লাইড
প্রথম স্লাইড তৈরি করো ও থিম বেছে নাও
তোমার স্কুলের বার্ষিক অনুষ্ঠানের প্রজেক্ট আছে। প্রথমবার PowerPoint খুলে একটি সুন্দর থিম দিয়ে শুরু করো।
▼
🏫
বাস্তব পরিস্থিতি: রিমা ক্লাস ৯-এর ছাত্রী। বিজ্ঞান প্রজেক্টের জন্য প্রথমবার PowerPoint খুলেছে। কোথা থেকে শুরু করবে বুঝতে পারছে না।
ধাপে ধাপে শেখো
  • PowerPoint খোলো → File → New → 'New Presentation' বা 'Blank Presentation' বেছে নাও।
  • থিম বেছে নিতে → উপরে Design ট্যাবে ক্লিক করো → অনেক থিম দেখতে পাবে।
  • রঙ বদলাতে → Design → Variants → Colors বেছে নাও।
  • সেভ করতে → Ctrl+S চাপো → ফাইলের নাম দাও → Save করো।
💡 টিপস
🎨সহজ ও পরিষ্কার থিম বেছে নাও — বেশি রঙিন থিম পড়তে কষ্ট হয়।
💾ফাইল .pptx ফরম্যাটে সেভ করো — এটা সব জায়গায় খোলা যায়।
⌨️ শর্টকাট
Ctrl+S সেভ
Ctrl+Z পূর্বাবস্থা
Ctrl+N নতুন ফাইল

Q 02 স্লাইড যোগ
নতুন স্লাইড যোগ করো ও লেআউট বেছে নাও
তোমার প্রেজেন্টেশনে ৫টি স্লাইড দরকার। প্রতিটিতে আলাদা লেআউট ব্যবহার করো।
▼
📚
বাস্তব পরিস্থিতি: করিম তার পরিবার পরিচিতির জন্য PPT বানাচ্ছে। প্রতিটি স্লাইডে আলাদা ধরন দরকার।
Title Slideশুধু শিরোনাম ও উপশিরোনামের জন্য
Title and Contentশিরোনাম + বুলেট পয়েন্ট
Two Contentদুই পাশে আলাদা বক্স
Blankসম্পূর্ণ ফাঁকা — নিজের ইচ্ছামতো সাজাও
ধাপে ধাপে শেখো
  • নতুন স্লাইড যোগ করতে → বাম পাশে ডান ক্লিক → New Slide।
  • লেআউট বদলাতে → Home → Layout বেছে নাও।
  • স্লাইড সরাতে → বাম প্যানেলে drag & drop করো।
  • স্লাইড কপি করতে → ডান ক্লিক → Duplicate Slide
📐'Title and Content' লেআউটটি সবচেয়ে বেশি ব্যবহার হয়।
🔢একটি প্রেজেন্টেশনে ৬–১০টি স্লাইডের বেশি না রাখাই ভালো।

Q 03 পটভূমি
স্লাইডের পটভূমির রঙ ও ছবি পরিবর্তন করো
তোমার বিজ্ঞান প্রেজেন্টেশনের শিরোনাম স্লাইডে আকাশের ছবি ব্যাকগ্রাউন্ড হিসেবে ব্যবহার করো।
▼
🌤️
বাস্তব পরিস্থিতি: নাফিসা 'মহাকাশ আবিষ্কার' নিয়ে প্রেজেন্টেশন বানাচ্ছে। প্রথম স্লাইডে তারার ছবি ব্যাকগ্রাউন্ড দিতে চায়।
  • শুধু রঙ দিতে: ডান ক্লিক → Format Background → Solid Fill
  • ছবি দিতে: Format Background → Picture or texture fill → Insert
  • ছবি উজ্জ্বল হলে → Transparency ৪০–৬০% করো।
  • সব স্লাইডে একসাথে দিতে → 'Apply to All' ক্লিক করো।
✍️ব্যাকগ্রাউন্ড গাঢ় হলে সাদা লেখা ব্যবহার করো।
🌈Gradient Fill দিয়ে দুটো রঙ মেশাও।
টেক্সট ও ফন্ট

Q 04 ফন্ট স্টাইল
শিরোনামে বড় ও আকর্ষণীয় ফন্ট ব্যবহার করো
তোমার প্রেজেন্টেশনের প্রথম স্লাইডে কোম্পানির নাম বড় ও সুন্দরভাবে লেখো।
▼
🏢
বাস্তব পরিস্থিতি: আরিফ তার নতুন ব্যবসার পরিচিতি স্লাইড বানাচ্ছে। শিরোনাম প্রথম দেখায় মনে রাখার মতো হওয়া দরকার।
  • Title বক্সে ক্লিক করো → লেখো → Ctrl+A দিয়ে সব সিলেক্ট।
  • ফন্ট বদলাতে: Home → Font বক্সে 'Georgia' বা 'Cambria' লেখো।
  • আকার বাড়াতে: Font Size বক্সে '44' বা '54' লেখো।
  • Bold: Ctrl+B · Italic: Ctrl+I · Underline: Ctrl+U
  • রঙ বদলাতে → Font Color (A চিহ্নের নিচে) ক্লিক করো।
📏শিরোনাম: ৩৬–৪৮pt। মূল লেখা: ২০–২৪pt।
🔠একটি স্লাইডে সর্বোচ্চ ২টি ফন্ট।
Ctrl+B বোল্ড
Ctrl+I ইটালিক

Q 05 বুলেট পয়েন্ট
পরিষ্কার বুলেট পয়েন্ট তৈরি করো
তোমার পণ্যের ৫টি বৈশিষ্ট্য স্লাইডে তালিকা আকারে সুন্দর করে লেখো।
▼
🍃
বাস্তব পরিস্থিতি: সুমাইয়া তার নতুন হার্বাল চায়ের ব্র্যান্ডের পরিচিতি দিচ্ছে। পণ্যের গুণাগুণ সহজে পড়ার মতো করে দেখাতে চায়।
  • Content বক্সে ক্লিক করো → লেখো → Enter দাও।
  • বুলেট স্টাইল বদলাতে → Home → Bullets ড্রপডাউন।
  • ৬×৬ নিয়ম: প্রতি বুলেটে ৬ শব্দ, স্লাইডে ৬টি পয়েন্ট।
  • লাইনের ফাঁকা বাড়াতে → Line Spacing → 1.5
📌বুলেটে পুরো বাক্য নয় — শুধু মূল শব্দ।

একটি ছবির উপরে বা নিচে তোমার পছন্দমতো জায়গায় একটি ক্যাপশন লেখো।

  • Insert → Text Box ক্লিক করো।
  • যেখানে চাও সেখানে মাউস দিয়ে বক্স তৈরি করো।

  • বক্স সরাতে → কিনারায় মাউস রেখে টেনে নিয়ে যাও।
  • বর্ডার দিতে → Shape Format → Shape Outline

তোমার পরিবারের একটি ছবি বা পণ্যের ছবি স্লাইডে যোগ করো ও সুন্দরভাবে সাজাও।
▼
🛒
বাস্তব পরিস্থিতি: মিতা পণ্য প্রেজেন্টেশন বানাচ্ছে। প্রতিটি স্লাইডে পণ্যের ছবি বাম দিকে ও বিবরণ ডান দিকে।
  • Insert → Pictures → This Device → ছবি বেছে নাও।
  • আকার বদলাতে → কোণায় Shift চেপে টানো।
  • ছবি সরাতে → মাঝখানে ক্লিক করো → টেনে নাও।
  • ছবি ছাঁটতে → Picture Format → Crop
  • বর্ডার বা ছায়া → Picture Styles থেকে বেছে নাও।

Q 11 স্পিকার নোট
প্রতিটি স্লাইডে স্পিকার নোট যোগ করো
