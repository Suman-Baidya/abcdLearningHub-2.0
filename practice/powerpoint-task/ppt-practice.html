Beginner Business 01
Company Introduction Deck
Create a 6-slide intro deck for a new startup. Cover mission, team, product, market, and contact.
▼
Real Scenario
🏢
Scenario: You've just launched "FreshBite" — a healthy meal delivery startup. Your first investor meeting is next Monday. You need a clean 6-slide deck to introduce the company to potential investors and partners.
Slide Structure Blueprint
Slide 1 — Title / Cover
📍 Slide 1 — Company name, tagline, logo
🎯 Slide 2 — Problem we solve
💡 Slide 3 — Our solution / product
📈 Slide 4 — Market opportunity
👥 Slide 5 — Team & credentials
📞 Slide 6 — Contact & call to action
Step-by-Step
  • Open PowerPoint → New Presentation → Choose a clean professional theme (try "Ion" or "Facet")
  • Slide 1: Click the title box → type company name in large bold font. Add a short tagline below. Insert your logo via Insert → Pictures
  • Use Slide Layout for each slide: Right-click slide → Layout → choose "Title and Content" or "Two Content"
  • Keep text minimal — max 3 bullet points per slide. Use the "6×6 rule": 6 words per bullet, 6 bullets max
  • Apply consistent color scheme: Design tab → Variants → choose accent colors matching your brand
  • Final check: View → Slide Sorter to see all 6 slides together and adjust flow
Pro Tips
🎨 Use only 2–3 brand colors. Investors trust consistency over creativity.
📏 Title font: 40–44pt. Body text: 20–24pt. Never go below 18pt.
🖼️ One high-quality image per slide beats 5 stock photos.
Keyboard Shortcuts
Ctrl+M New slide
Ctrl+D Duplicate slide
F5 Present from start
Ctrl+Z Undo
Intermediate Business 02
Quarterly Sales Report
Build a Q3 sales performance deck with KPIs, charts, regional breakdown, and next quarter targets.
▼
Real Scenario
📊
Scenario: You're the Sales Manager at a retail company. Every quarter, you present performance data to the board. Create a Q3 report deck showing what worked, what didn't, and what's planned for Q4.
Slide Structure
📌 Slide 1 — Q3 Summary cover
🔢 Slide 2 — KPI Scorecard (Revenue, Units, NPS)
📈 Slide 3 — Monthly revenue bar chart
🗺️ Slide 4 — Regional performance table
🏆 Slide 5 — Top 5 products
🎯 Slide 6 — Q4 targets & action plan
Step-by-Step
  • Slide 2 KPI cards: Insert → Shapes → Rectangle. Create 3 boxes side by side. In each, add the KPI name in small text and the number in large bold. Group them.
  • Slide 3 Chart: Insert → Chart → Bar Chart → Clustered Bar. Edit the data table that appears — type in Month and Revenue columns.
  • Format chart: Remove gridlines (click them → Delete). Change bar color to match theme. Add Data Labels (right-click bars → Add Data Labels).
  • Slide 4 Table: Insert → Table → 5×4. First row = header. Use Table Design tab → choose a "Medium Style" preset.
  • Add conditional arrows: Insert → Icons → search "up arrow" / "down arrow" to show growth/decline visually next to numbers.
Pro Tips
🔴 Use red/green color coding for below/above target — universal and instantly understood.
📐 Align all elements: Select multiple → Shape Format → Align → Align Center.
Beginner Education 03
School Science Project Presentation
Create a 5-slide science fair presentation about the water cycle with visuals and labeled diagrams.
▼
Real Scenario
🌊
Scenario: Class 8 student Rina needs to present her science project on "The Water Cycle" to her class. She has 5 minutes and wants a visually rich, easy-to-follow presentation that earns top marks.
Slide Structure
🌍 Slide 1 — Title with water image background
💧 Slide 2 — What is the water cycle?
☁️ Slide 3 — Diagram with labels (SmartArt)
🔬 Slide 4 — Key stages explained
✅ Slide 5 — Fun facts + conclusion
Step-by-Step
  • Slide 1 background: Click slide → Design → Format Background → Picture or texture fill → insert a water/sky image. Set transparency to 60% so text stays readable.
  • Add a semi-transparent title box: Insert → Shape → Rectangle → set Fill to black 40% transparency. Type title on top.
  • Slide 3 SmartArt: Insert → SmartArt → Cycle → Basic Cycle. Click each box to type: Evaporation → Condensation → Precipitation → Collection → Transpiration.
  • Add callout arrows: Insert → Shapes → Callout shapes to add labels pointing to parts of the diagram.
  • Slide 4: Use Two Content layout — put stage name + icon on left, explanation on right.
Pro Tips
🌈 Use a "sky blue" theme — it matches the topic and makes diagrams feel natural.
🎤 Add Speaker Notes (View → Notes) to remember what to say for each slide.
Intermediate Design 04
Master Slide & Consistent Branding
Set up a Slide Master with your company logo, brand fonts, colors, and footer on every slide automatically.
▼
Real Scenario
🏷️
Scenario: You work at "GreenTech Solutions." Every department makes their own slides — but they all look different! Your boss wants a unified company template so all 50-slide decks look professional and on-brand.
Step-by-Step
  • Go to View → Slide Master. The first large slide is the "parent" — changes here affect all layouts below it.
  • Click the parent slide. Insert → Pictures → add company logo. Resize to ~1cm and drag to the bottom-left corner.
  • Set brand font: In Slide Master view → Slide Master tab → Fonts → Customize Fonts. Set Heading and Body fonts.
  • Set brand colors: Slide Master tab → Colors → Customize Colors. Set all 10 color slots to match your brand palette.
  • Edit the footer area at the bottom — add company name or slide number placeholder via Insert → Header & Footer.
  • Close Master View: Slide Master tab → Close Master View. Now every new slide inherits your branding.
Pro Tips
💾 Save as .potx (PowerPoint Template) so anyone in the company can use it as a starting file.
🔒 Lock background elements in Master so presenters can't accidentally delete the logo.
Beginner Animation 05
Animated Product Launch Slide
Use entrance animations to reveal a product name, tagline, and image one by one for a dramatic reveal effect.
▼
Real Scenario
🚀
Scenario: You're launching "ZenBrew" — a new herbal tea brand at a trade fair. You need one powerful slide that animates your product name and image on screen dramatically when you click, creating a "wow" moment.
Step-by-Step
  • Create a dark background slide. Type the product name "ZenBrew" in large text (80pt). Insert product image to the right.
  • Click the product image → Animations tab → Add Animation → Entrance → Zoom. This makes it zoom in on click.
  • Click the title text → Animations → Add Animation → Fly In. Set Direction to "From Left".
  • Open Animation Pane (Animations tab → Animation Pane). Reorder: image first, then title. Set title to start "After Previous" (auto after image).
  • Add tagline text below title → Animate with Fade → Start: "After Previous" → Duration: 1.5 sec.
  • Test: Press F5 and click once — watch the reveal sequence.
Pro Tips
⚡ Keep animation duration 0.5–1 sec. Anything slower feels unprofessional in live presentations.
🚫 Never use "Bounce", "Spiral" or "Swivel" in business settings — they're distracting.
Shortcuts
Alt+A Animations tab
Shift+F5 Present current slide
Intermediate Data & Charts 06
Sales Performance Bar Chart Slide
Build an impactful bar chart showing 5 products' sales, formatted with data labels, clean axes, and color highlights.
▼
Real Scenario
📦
Scenario: As Product Manager at a grocery chain, you need to show which of 5 product categories performed best this year. Your chart must be immediately readable by busy executives in less than 5 seconds.
Step-by-Step
  • Insert → Chart → Bar → Clustered Bar. An Excel-like datasheet opens. Replace sample data with: Products (Dairy, Produce, Snacks, Beverages, Frozen) and their revenue values.
  • Close the data window. Click the chart → Chart Design tab → Change Colors → pick a monochromatic single-color scheme for clean look.
  • Highlight the top performer: Click one bar (to select all) → click it again (to select just that one) → change fill to a bright accent color.
  • Add data labels: Right-click bars → Add Data Labels. Then right-click labels → Format Data Labels → check "Value", uncheck "Category".
  • Remove clutter: Click gridlines → Delete. Click Y-axis numbers → Delete. Click chart border → set to No Line.
  • Add a descriptive title above the chart. Use a bold number callout: e.g., "Dairy: +32% YoY" in a text box near the highlighted bar.
Pro Tips
🎯 Sort bars from highest to lowest — it's easier to read rankings visually.
🔢 Round all numbers to 1 decimal. ₹1.2M is cleaner than ₹1,234,567.
Advanced Business 07
Investor Pitch Deck (10 Slides)
Build a complete startup pitch following the Guy Kawasaki 10-slide rule with problem, solution, business model, and ask.
▼
Real Scenario
💰
Scenario: You're pitching "EduBot AI" — an AI tutoring app — to a panel of 3 venture capitalists. You have exactly 10 minutes. Following Guy Kawasaki's rule: 10 slides, 20 minutes, 30pt font minimum.
The 10-Slide Structure
1️⃣ Title — Company, name, contact
2️⃣ Problem — The pain point
3️⃣ Solution — Your product
4️⃣ Business Model — How you earn
5️⃣ Go-to-Market — Customer strategy
6️⃣ Competition — Your advantage
7️⃣ Team — Who you are
8️⃣ Projections — 3-year financials
9️⃣ Status & Timeline
🔟 The Ask — Funding amount
Step-by-Step
  • Slide 6 Competition Matrix: Insert → Table → 5 rows × 4 cols. Rows = competitors, Cols = key features. Use ✓ and ✗ symbols.
  • Slide 8 Financial chart: Insert → Chart → Line Chart. Show Year 1–3 projected revenue. Add a "Break-Even" annotation line.
  • Use Morph transition between slides for a premium flowing feel: Select all slides → Transitions → Morph.
  • Slide 10 "The Ask": Make it bold and simple. One large number (e.g., "$500K"), below it: "For 10% equity". Nothing else.
Pro Tips
📏 Kawasaki Rule: minimum 30pt font. If you need smaller text, you have too much content.
🖼️ Every slide should have ONE key message. If you can't say it in one sentence, cut it.
Beginner Data & Charts 08
Budget Breakdown Pie Chart
Create a department budget pie chart with labels, percentages, an exploded slice for the largest category.
▼
Real Scenario
💼
Scenario: Finance manager Anika presents the annual departmental budget to the leadership team. She needs a pie chart showing the % share of each department: HR 15%, IT 30%, Marketing 20%, Operations 25%, R&D 10%.
Step-by-Step
  • Insert → Chart → Pie → Pie. In the data table: Column A = Department names, Column B = percentages.
  • Explode the largest slice (IT 30%): Click the chart → click the IT slice → drag it outward slightly to "explode" it.
  • Add labels: Right-click → Add Data Labels → Format Data Labels → check Category Name + Percentage. Uncheck Value.
  • Change colors: Click each slice and change fill to distinct but harmonious colors from your theme palette.
  • Add a callout: Insert → Shapes → Oval Callout → point at IT slice → type "Largest spend: IT Infrastructure".
Pro Tips
🥧 Pie charts work best with 5 or fewer slices. More than 6 → use a bar chart instead.
↗️ Start the largest slice at the 12 o'clock position for natural reading order.

Teacher's Lesson Plan Presentation
Design an engaging class lesson on "Climate Change" with activities, discussion questions, and timed segments.
▼
Real Scenario
🌿
Scenario: Geography teacher Mr. Hassan has a 45-minute class on Climate Change. He needs slides that not only present information but actively engage students with activities, polls, and discussion breaks.
Slide Structure
⏱️ 0:00 — Hook: Shocking stat image
📖 0:05 — Learning objectives
🌍 0:10 — Core content (causes)
🗣️ 0:25 — Think-pair-share activity
📊 0:35 — CO₂ data trend chart
✅ 0:42 — Summary + homework
Step-by-Step
  • Slide 1 Hook: Find a powerful before/after glacier image. Set as full-bleed background. Add minimal text overlay — just a statistic: "Arctic ice has lost 13% per decade."

  • Activity slide: Use a bright yellow/green background to visually signal "different mode." Add a large timer graphic via Shapes.

  • Insert a hyperlink button on slide 5: Insert → Shapes → Action Button → "Forward" to jump to the homework slide if time runs short.
Pro Tips
🎨 Use a different background color for "activity" slides vs "content" slides — students know when to engage.
🔗 Embed a QR code linking to a Kahoot quiz for instant formative assessment at the end.
Intermediate Design 10
Professional Photo Portfolio Deck
Create a stunning portfolio of 8 project images with captions, consistent layout, and smooth transitions.
▼
Real Scenario
📷
Scenario: Freelance architect Priya is meeting a new client. She wants to show her 8 best completed projects in a sleek portfolio presentation that she can also email as a PDF.
Step-by-Step
  • Set slide size to Widescreen 16:9. Go to Design → Slide Size → Widescreen. Use a dark or white background for maximum image contrast.
  • Consistent image layout: Insert image → hold Shift and drag corners to resize proportionally. Use Align tools to position identically on each slide.
  • Crop to consistent ratio: Click image → Picture Format → Crop → Aspect Ratio → 16:9 for all images.
  • Add a semi-transparent caption bar: Insert → Rectangle → set to black with 50% transparency → place at bottom of image → add project name text over it.
  • Apply Fade transition to all slides: Select all slides → Transitions → Fade → Duration 0.5s. This creates an elegant gallery feel.
  • Export as PDF: File → Export → Create PDF/XPS. This is perfect for emailing as a viewable portfolio.
Pro Tips
📐 Use Picture Layout (Format → Picture Layout) to auto-arrange multiple images in a grid cleanly.
🖼️ Compress images before inserting: Picture Format → Compress Pictures → Email (96ppi) to keep file size manageable.
Advanced Animation 11
Morph Transition Story Timeline
Use Morph transitions to animate a company timeline that smoothly flows from year to year with morphing elements.
▼
Real Scenario
🕰️
Scenario: At a company's 10th anniversary celebration, you need to show a visual timeline of key milestones (2015–2025) that animates smoothly as you click through — like a movie, not a boring slide show.
Step-by-Step
  • Slide 1: Draw a horizontal line across the slide. Place a circle at the far left labeled "2015". Add text box above: "Company Founded".
  • Duplicate this slide. On Slide 2, move the year circle to the next position (2017). Change its label. Add a new milestone text box. Keep the line in the same position!
  • Apply Morph transition to Slide 2: Transitions → Morph. PowerPoint automatically animates the circle sliding from 2015 to 2017 position.
  • Repeat: duplicate each slide, move the circle forward, add a new milestone. Apply Morph to each new slide.
  • Add milestone icons: Insert → Icons → search relevant icons (rocket, award, globe) → place near each year on the appropriate slide.
  • Optional: add a progress bar at the bottom that grows longer with each slide using the same Morph technique.
Pro Tips
🔑 Morph works by matching object names. Rename objects: right-click → Edit Alt Text → give same name on both slides.
✨ Use Morph to animate zoom-ins — place the same image small on slide 1, large on slide 2. Morph makes it zoom smoothly.

Intermediate Data & Charts 12
Survey Results Infographic Slide
Turn raw survey data into a visual infographic slide with icon arrays, percentages, and comparison bars.
▼
Real Scenario
📋
Scenario: HR ran an employee satisfaction survey — 200 responses. Results: 68% satisfied with work-life balance, 42% want more remote work, 55% feel recognized. You need one compelling infographic slide for the all-hands meeting.
Step-by-Step
  • Create a 3-column layout using text boxes. Each column = one survey question + result.
  • For "68% satisfied": Insert a large text box → type "68%" in 80pt bold. Below it, add the question in 14pt. Then draw a thin horizontal bar using rectangles: a full-width background bar + a colored 68%-width bar on top.
  • Icon array for "42% want remote": Insert 10 person icons (Insert → Icons → person). Color 4 of them blue (representing 42%). Group them. Add label "4 in 10 employees".
  • For "55% feel recognized": Use a Donut chart (Insert → Chart → Doughnut) with 55% and 45% as data. Make the center hole large, and add "55%" as a text box in the center.
  • Add a headline at the top: "What Our Team Is Telling Us" in bold Fraunces/Playfair style font.
Pro Tips
👥 Icon arrays (people icons) are more human and impactful than bar charts for survey data.
💡 Highlight the most surprising stat with a bright color accent — it draws the eye immediately.
Beginner Business 13
Project Status Report Slide
Build a weekly project status update with RAG (Red/Amber/Green) status indicators, progress bars, and blockers.
▼
Real Scenario
🚦
Scenario: Project manager Farhan reports to the steering committee every Monday. He needs a single-slide "dashboard" showing 5 workstreams, their status (on track / at risk / delayed), and % completion — readable in 30 seconds.
Step-by-Step
  • Create a table: Insert → Table → 6 rows × 4 cols. Columns: Workstream | Owner | % Complete | Status.
  • For the "% Complete" column: Add a thin background rectangle in each cell, then a colored rectangle on top representing the percentage width. Group each pair.
  • For "Status" column: Insert → Shapes → Oval. Create 3 small circles — one red, one amber, one green — and copy the appropriate one into each row.
  • Add a Summary box at the top: 3 large colored circles with numbers: "3 On Track | 1 At Risk | 1 Delayed".
  • Date/sprint label: Add "Week of [date]" in the header. This makes it reusable — just change the date each week.
Pro Tips
🚦 RAG (Red/Amber/Green) is universally understood in business. Use exact colors: #cc0000 / #e6a817 / #009933.
📌 Keep this template saved — copy it each week and just update the numbers. 2 minutes to update.
Advanced Design 14
Full-Bleed Visual Impact Slide
Design a full-screen image slide with text overlay, gradient masks, and typographic hierarchy for maximum visual punch.
▼
Real Scenario
🌅
Scenario: You're opening a leadership conference keynote. The first slide needs to stop the room — a full-screen powerful image with a single bold statement that sets the emotional tone before you say a word.
Step-by-Step
  • Right-click slide → Format Background → Picture or Texture Fill → Insert image. Set "Offset" all to 0 for perfect full-bleed. Darken image using "Picture Corrections" → Brightness -20%.
  • Add a gradient overlay: Insert → Rectangle (full slide size) → Fill: Gradient → Linear → Angle 90° → Left stop: Black 70% opacity, Right stop: Black 0% opacity.
  • Add headline text in massive font (80–100pt). Use a light/white color. Position in lower-left for modern editorial feel.
  • Add a colored accent line: Insert → Line shape → make it 3px thick in brand color → place just above the headline.
  • Sub-headline in 24pt with 60% white opacity below the main headline. Conference name/date in 14pt at very bottom.
Pro Tips
📸 The best images have a clear subject with open space for text — avoid busy, cluttered images.
✏️ Never put text on the brightest part of an image. Use the gradient overlay to darken the text area.
Beginner Education 15
College Thesis Defense Slides
Build a formal 12-slide academic thesis presentation with abstract, methodology, findings, and recommendations.
▼
Real Scenario
🎓
Scenario: Final-year student Nadia is defending her thesis "Impact of Social Media on Youth Mental Health" before a 5-member academic panel. She has 20 minutes and needs a formal, credible, well-structured presentation.
Structure
📌 Title, name, institution, date
📖 Research Overview & Abstract
🔬 Methodology (sample, tools)
📊 Key Findings (3 charts)
🗣️ Discussion & Implications
✅ Recommendations & Limitations
Step-by-Step
  • Use a formal serif theme: Design → Browse Themes → or apply a neutral Academic theme. Avoid flashy colors.
  • Title slide: University logo top-left. Thesis title in 36pt center. Your name, supervisor, department, date in 18pt.
  • Methodology slide: Use SmartArt → Process → Basic Chevron Process to show your research steps visually.
  • Findings slides: One chart per finding. Insert chart → choose appropriate type. Add a "Key Takeaway" box at the bottom of each chart slide in a colored rectangle.
  • Add slide numbers: Insert → Header & Footer → check Slide Number → Apply to All.
Pro Tips
🎓 Academic rule: no more than 7 words per bullet point. Elaborate verbally — the panel reads faster than you speak.
📑 Prepare backup slides after slide 12 with detailed tables/data for panel Q&A — insert them but they won't show in normal flow.
Intermediate Business 16
Comparison / Before-After Slide
Design a two-column comparison slide showing old process vs new solution, problem vs outcome, or competitor vs you.
▼
Real Scenario
⚖️
Scenario: You're a consultant proposing a new HR system. You need one slide that vividly shows: "Current State (Manual, slow, error-prone)" vs "Future State (Automated, fast, accurate)" — compelling enough for the CFO to approve the budget.
Step-by-Step
  • Split the slide into two halves: Insert two large rectangles side by side. Left = grey/muted color ("Current State"). Right = your brand color ("Future State").
  • Label each half: "BEFORE" and "AFTER" in large text at the top of each box, or "Manual Process" vs "Automated System".
  • Add bullet points on each side with icons. Left side icons: ❌ 3 days to process / ❌ 12% error rate. Right side: ✅ 2 hours / ✅ 0.1% error rate.
  • Add a vertical dividing line with a VS icon or arrow in the center using Insert → Shapes.
  • At the bottom, add a summary callout: "Save 200 hours/month + ₹4.5L in operational costs" in a highlighted box.
Pro Tips
🎭 The visual contrast (dull grey vs vibrant color) subconsciously tells the story before they read a word.
💰 Always quantify the "after" — executives make decisions based on numbers, not descriptions.
Advanced Data & Charts 17
Multi-Chart Dashboard Slide
Create an executive dashboard with 4 mini-charts (line, bar, donut, KPI card) arranged on a single slide.
▼
Real Scenario
📱
Scenario: The CEO wants a single "one-pager" slide each Monday with the company's health metrics — revenue trend, customer count, churn rate, and NPS score — all visible at a glance without scrolling.
Step-by-Step
  • Divide slide into a 2×2 grid using guide lines: View → Guides → check. Drag guides to create 4 equal quadrants.
  • Top-left: Insert → Chart → Line → Monthly Revenue Trend (12 months). Remove all chrome except the line and axis labels.

Pro Tips
🎯 Use the same color palette for all 4 charts — visual consistency makes a dashboard feel "designed" not cobbled.
🔢 Every chart needs a "so what" label — not just "Revenue" but "Revenue: +12% vs last month ✅".
Intermediate Design 18
SmartArt Process Flow Diagram
Build a visual process flow for onboarding, a sales funnel, or a supply chain using SmartArt and custom styling.
▼
Real Scenario
🔄
Scenario: Operations Manager Sanjay is presenting a new employee onboarding process to HR. He needs a clear visual showing the 6 stages from offer letter to 90-day check-in, with timing and responsibilities.
Step-by-Step
  • Insert → SmartArt → Process → Accent Process. This gives you chevron-shaped steps perfect for a flow.
  • Click each chevron and type the stages: "Offer Letter → Pre-boarding → Day 1 → Week 1 → Month 1 → 90-Day Review".
  • Customize colors: Click SmartArt → SmartArt Design → Change Colors → Colorful Accent Colors.

Pro Tips
🔄 Always convert SmartArt to shapes before final formatting — SmartArt auto-adjusts can fight your design choices.
🎨 Use progressively darker shades of one color to show time passing — light → dark = past → future.
Intermediate Animation 19
Animated Build — Reveal Points One by One
Animate a 5-point argument slide so each bullet appears on click, keeping the audience focused on your current point.
▼
Real Scenario
🗣️
Scenario: You're presenting 5 reasons why your company should adopt remote work policy. If you show all 5 at once, people read ahead and stop listening. Use "build" animation to reveal each reason only when you're ready.
Step-by-Step
  • Type all 5 bullet points in a text box on the slide. They should all be visible while editing.
  • Click the text box → Animations → Add Animation → Entrance → Appear.
  • Open Animation Pane. You'll see 5 animation entries. Click each one → Effect Options → "By 1st Level Paragraphs" — this splits it into one animation per bullet.
  • Set each to "On Click" trigger. Now each click reveals the next bullet point.
  • Optional dimming effect: After each bullet is revealed, the previous one dims. Click a bullet animation → Effect tab → After animation: change color to light grey.
Pro Tips
🎯 The dimming effect is a top TED-style technique — current point stays bright, past points fade — keeping focus.
⌨️ In Presenter View, you see all bullets while audience only sees revealed ones — a huge presenting advantage.
Advanced Education 20
Interactive Hyperlinked Presentation
Build a non-linear menu-driven presentation where clicking topic buttons jumps to different sections — like a website.
▼
Real Scenario
🖱️
Scenario: Training manager Deepa creates an onboarding presentation for new employees. Rather than linear slides, she wants a main menu where users click any topic (Company Culture / Benefits / IT Setup / Policies) and jump directly to that section.
Step-by-Step
  • Build your sections first: Create Slide 1 (Menu), then slides 3–6 for each section, with a "Home" button that returns to the menu.
  • Menu slide: Create 4 large clickable buttons using shapes. Label them: "Company Culture", "Benefits", "IT Setup", "Policies".
  • Add hyperlink to each button: Right-click shape → Link → This Document → select target slide. Click OK.
  • On each section's first slide, add a small "← Menu" button in the corner: same process, link it back to Slide 1.
  • Add a consistent navigation bar at the bottom of every slide: small buttons for each section using the same hyperlink method.
  • Test in Slide Show mode (F5): click each button to verify all links work correctly.
Pro Tips
🖱️ Use "hover" action (Insert → Action → Mouse Over → Hyperlink) for tooltip-style interactivity on menu buttons.
📱 This technique creates a self-navigating kiosk-style presentation — perfect for trade fair booths.

B Black screen during show
