CREATING A NEW DOCUMENT

1
File → New (Ctrl+N)

Opens the New Document dialog. Choose from preset categories: Photo, Print, Web, Mobile, Film & Video, or enter custom dimensions.

2
Set Width, Height & Resolution

For print: set in cm/mm, 300 DPI. For web/screen: set in pixels, 72-96 PPI. Color mode: RGB for screen, CMYK for professional print.

3
Set Color Mode & Bit Depth

RGB Color, 8-bit for standard work. 16-bit for advanced color editing. CMYK for offset printing. Never work in Indexed Color.

4
Set Background Contents

White (most common), Background Color, or Transparent (for graphics with no background). Click Create.

ESSENTIAL PREFERENCES

Go to Edit → Preferences (Ctrl+K) on PC, or Photoshop → Preferences on Mac. Key settings to change:

💾 Performance

Set RAM usage to 70–80%. Assign fast SSD as Scratch Disk #1. Enable GPU acceleration. These are the most impactful performance settings.

🎨 Interface

Choose UI color theme (Dark/Medium Gray recommended). Enable "Show Tool Tips." Set OpenGL options for better canvas performance.

📁 File Handling

Enable "Auto Save Recovery Info Every: 5 Minutes." This can save you from losing hours of work during crashes.

🖱️ Cursors

Set Painting Cursors to "Full Size Brush Tip" and "Show Crosshair in Brush Tip." This shows exact brush size while painting.

ℹ️
Workspace Tip: Go to Window → Workspace → Photography or Graphic and Web to switch between preset panel layouts optimized for different tasks. Save your own layout with Window → Workspace → New Workspace.

BRUSH SETTINGS MASTERY

The Brush tool is the most powerful tool in Photoshop. Open the Brush Settings panel via Window → Brush Settings (F5) for complete control.

📏 Size & Hardness

Use [ and ] keys to decrease/increase brush size. Shift+[ / Shift+] changes hardness. 0% = soft feathered edge. 100% = hard crisp edge.

📐 Shape Dynamics

Size Jitter, Angle Jitter — creates natural randomness. Control: "Pen Pressure" with a drawing tablet for pressure-sensitive painting.

🌫️ Scattering

Makes brush marks scatter around the stroke path. Use for stars, foliage, particles. Adjust Count and Count Jitter for density.

💧 Flow vs. Opacity

Opacity controls overall transparency. Flow controls paint buildup per stroke. Low flow = airbrushing effect. Both at 100% = solid opaque strokes.

LAYERS

The single most important concept in Photoshop. Layers let you work non-destructively, stack elements, and control every part of your design independently.

UNDERSTANDING LAYERS

Think of layers as transparent sheets stacked on top of each other. Each layer contains different elements. You can edit, move, hide, or delete any layer without affecting others.

LAYER TYPES

📷 Pixel Layer

Regular image layer containing rasterized pixels. Most common. Editing is destructive — enlarging loses quality.

🔤 Type Layer

Vector-based text layer. Always editable. Keeps text crisp at any size. Rasterize only when necessary.

🔷 Shape Layer

Vector shape — scales without quality loss. Can edit anchor points with Direct Selection tool.

🎛️ Adjustment Layer

Non-destructive edits (curves, levels, hue) that affect all layers below. Can be toggled off, edited, or deleted.

📦 Smart Object

Embedded or linked content that retains original quality. Filters applied to Smart Objects are non-destructive.

📁 Group Layer

Folder containing multiple layers. Organize complex designs. Can apply blend modes and masks to entire groups.

BLEND MODES EXPLAINED

Blend modes control how a layer interacts with the layers below it. Most used modes:

MULTIPLY
Darkens. White disappears. Used for shadows, textures on light backgrounds. Multiplies color values.
SCREEN
Brightens. Black disappears. Used for glows, fire, fog effects on dark backgrounds.
OVERLAY
Increases contrast. Midtones transparent. Used to add texture while preserving image detail.
SOFT LIGHT
Subtle Overlay. Gentler contrast boost. Great for skin retouching and subtle color grading.
LUMINOSITY
Applies brightness without color shift. Perfect for sharpening layers without color halos.
COLOR
Applies hue/saturation without changing luminosity. Use to colorize grayscale images.

LAYER STYLES (FX)

Double-click any layer to open Layer Style dialog. These are non-destructive effects:

🌑 Drop Shadow

Adds shadow behind element. Control angle, distance, size, opacity. Essential for making text readable over photos.

✨ Outer Glow

Glowing halo around element edge. Used for neon effects, highlighted text, magical effects.

🔲 Stroke

Adds border around layer. Inside, Outside, or Center. Use for text outlines, bordered shapes.

🎨 Color Overlay

Fills entire layer with a color at specified opacity. Quick way to change element color without destructive edit.

🌈 Gradient Overlay

Fills layer with gradient. Control style (linear, radial, angle), angle, and scale. Powerful for buttons, backgrounds.

🧱 Bevel & Emboss

3D raised or carved appearance. Used for metallic text, buttons, logo treatments.

✦
Layer Mask vs. Eraser: Never use the Eraser tool to remove backgrounds. Always use a Layer Mask — it's non-destructive. You can paint back hidden areas with white paint on the mask at any time. The Eraser permanently deletes pixels and can never be recovered.

SELECT & MASK — THE PROFESSIONAL WAY

1
Make an initial rough selection

Use Quick Selection (W) or Select Subject to get a rough selection of your subject. It doesn't need to be perfect — Select & Mask will refine it.

2
Open Select & Mask

Go to Select → Select and Mask (Alt+Ctrl+R). The workspace opens with your selection visible.

3
Use Refine Edge Brush on Hair/Fur

Select the Refine Edge Brush tool (R) and paint over the hair/fur edges. Photoshop AI detects fine strands automatically.

4
Adjust Edge Detection Settings

Smart Radius: ON. Radius: 2–5px for hard edges, 10–25px for hair. Smooth: 2–3. Feather: 0.5–1px. Contrast: 10–25%.

5
Set Output

Output To: "New Layer with Layer Mask" → OK. Your subject is now on its own layer with a professional mask.

🏢 Real-Life Example
Scenario: Product Photography Background Removal

An e-commerce seller photographs sarees on a mannequin against a grey background. Using Select Subject → Select & Mask, they remove the background in under 2 minutes per image, place products on a clean white background, and deliver 50+ product images per day. The Pen Tool would give perfection for jewellery photos where every edge must be pixel-perfect.

PHOTO RETOUCHING

Professional skin retouching, blemish removal, frequency separation, body reshaping, and object removal — the techniques used by professional photographers.

BLEMISH & SPOT REMOVAL

1
Duplicate the Background Layer

Always retouch on a duplicate. Ctrl+J. Name it "Retouch." This keeps the original intact.

2
Spot Healing Brush (J)

Make sure "Content-Aware" is selected in Options Bar. Set brush size slightly larger than the blemish. Single click. Photoshop automatically samples surrounding skin texture.

3
Healing Brush for Complex Areas

Alt+Click to manually sample clean skin texture, then paint over the problem area. More control than Spot Healing for areas near edges (eyes, lips).

4
Patch Tool for Large Areas

Draw around the area to fix → drag the selection to a clean texture area. Best for removing large patches or objects.

FREQUENCY SEPARATION (Professional Skin)

Frequency Separation separates skin texture (high frequency) from color/tone (low frequency) so you can retouch each independently without destroying natural skin texture.

1
Duplicate Background Twice

Layer 1 (Low frequency) — rename "Low Freq". Layer 2 (High frequency) — rename "High Freq". Turn off High Freq visibility.

2
Apply Gaussian Blur to Low Freq

Select Low Freq layer. Filter → Blur → Gaussian Blur. Set radius to 3–8px (enough to blur all skin detail). Click OK.

3
High Pass on High Freq

Select High Freq layer (turn visibility on). Image → Apply Image. Set: Layer: Low Freq, Blending: Subtract, Scale: 2, Offset: 128. OK. Then change blend mode to Linear Light.

4
Retouch Each Frequency Independently

Low Freq: Use Clone Stamp or Healing Brush for uneven skin tone, shadows, color blotches. High Freq: Use Clone Stamp to replace/move skin texture without affecting color.

⚠️
Over-Retouching Warning: The goal of skin retouching is to make skin look naturally better, not plastic or fake. Preserve at least 70% of original texture. If skin looks like glass or rubber, you've gone too far. Zoom out to 25% to judge the overall result.

CONTENT-AWARE FILL (Object Removal)

Remove unwanted objects, people, watermarks, and distractions from photos using Photoshop's AI-powered Content-Aware Fill.

1
Select the object to remove

Use Lasso or Quick Selection tool to make a loose selection around the object. Add 5–10px of surrounding area in your selection for better results.

2
Edit → Content-Aware Fill

The Content-Aware Fill workspace opens. You'll see a live preview of the removal on the right.

3
Exclude areas if needed

Use the Sampling Brush to paint green (include) or red (exclude) over areas to guide which textures Photoshop uses to fill.

4
Output to New Layer

Choose "New Layer" in Output Settings → OK. Keeps your original intact. Use Healing Brush to clean up any remaining imperfections.

🏢 Real-Life Example
Scenario: Wedding Photography Retouching in West Bengal

A photographer from Kolkata shoots wedding portraits. Using Frequency Separation on skin ensures the bride's natural complexion looks flawless without the plastic look. Content-Aware Fill removes a stray cord or uninvited guest from the background shot. Dodge & Burn creates dimension by lightening forehead highlights and darkening shadow areas around the jawline. Final export: 300 DPI JPG for album printing, 72 DPI JPG for WhatsApp/social media.

Project 5.1

Portrait Retouching Workflow

  1. Open a portrait photo and duplicate background (Ctrl+J)
  2. Use Spot Healing Brush to remove 5 blemishes

CURVES — THE MOST POWERFUL TOOL

Curves (Ctrl+M or Adjustment Layer) is the most precise color and tonal control in Photoshop. The diagonal line = tonal range from black (bottom-left) to white (top-right).

Curves Techniques
# S-CURVE: Classic contrast boost
Upper right point:  Drag UP   → Brighten highlights
Lower left point:   Drag DOWN → Deepen shadows
Result: Increased contrast + vibrance

# BRIGHTEN MIDTONES:
Center point (128): Drag UP → Brightens overall without blowing highlights

# FIX COLOR CAST: (e.g., remove yellow cast)
Channel: Blue    → Drag midpoint UP slightly
Channel: Red     → Drag midpoint DOWN slightly

# CINEMATIC FADE (film look):
Bottom-left point: Drag UP to RGB: (0, 30) → lifts blacks/shadows
Top-right point:   Drag DOWN to RGB: (255, 225) → reduces highlights
Gives a faded, analog film aesthetic

# TEAL & ORANGE LOOK (cinema grade):
Blue channel:    Lift shadows UP (adds blue/teal to darks)
Red channel:     Push highlights UP (adds orange warmth to lights)

LEVELS vs. CURVES

📊 Levels (Ctrl+L)

3 sliders: Shadows (black point), Midtones (gamma), Highlights (white point). Simpler than Curves. Use Levels to quickly fix exposure and set black/white points.

📈 Curves (Ctrl+M)

Infinitely flexible tonal control. Can create multiple control points. Adjust each color channel independently. Use Curves for all serious color work.

🎨 Hue/Saturation (Ctrl+U)

Control hue, saturation, and lightness. Target specific colors (Reds, Yellows, Greens). Colorize grayscale images. Quickly fix over-saturated skin or grass.

🌡️ Color Balance

Adjust shadows, midtones, highlights separately by pushing toward Cyan/Red, Magenta/Green, Yellow/Blue. Quick way to warm up or cool down shadows.

CAMERA RAW FILTER

Camera Raw Filter (Filter → Camera Raw Filter, Shift+Ctrl+A) gives you Lightroom-level editing inside Photoshop. Apply it as a Smart Filter to keep it editable.

☀️ Basic Panel

Exposure, Contrast, Highlights, Shadows, Whites, Blacks, Clarity, Dehaze, Vibrance, Saturation. This alone handles 80% of photo editing.

🎨 HSL Panel

Hue, Saturation, Luminance per color. Change grass from yellow-green to rich green. Darken blue sky. Fix orange skin tones.

🌃 Tone Curve

Point Curve (same as Curves in PS) plus Parametric curve. Color grade with Highlights, Lights, Darks, Shadows sliders per channel.

🏢 Real-Life Example
Scenario: Commercial Food Photography Edit

A Dhaka-based food photographer shoots biryani for a restaurant client. The image looks dull and slightly cool (LED lighting). Workflow: Camera Raw Filter → push Exposure +0.4, Shadows +40, Clarity +20 → HSL: Reds more saturated for meat, Yellows more saturated for rice → Curves: warm S-curve, push Red channel highlights up for a warm golden tone → Vibrance +15. Result: mouth-watering food photo that makes you hungry.

Project 6.1

Cinematic Photo Color Grade

  2. Convert background to Smart Object (right-click → Convert to Smart Object)
  3. Apply Camera Raw Filter: set Exposure, bring up Shadows, lower Highlights

MASKS & CUTOUTS

Layer masks, Clipping masks, vector masks — the non-destructive way to hide, reveal, and composite elements professionally.

LAYER MASK FUNDAMENTALS

A Layer Mask is a grayscale channel attached to a layer. White = Visible, Black = Hidden, Gray = Partially visible. Painting with black on a mask hides. Painting with white reveals.

➕ Adding a Layer Mask

Select layer → click □ (mask icon) at bottom of Layers panel. Or Layer → Layer Mask → Reveal All (white/show all) or Hide All (black/hide all).

🖌️ Painting on Mask

Click the mask thumbnail (not the layer thumbnail). Paint with Black brush to hide, White brush to reveal. Press X to swap foreground/background colors.

🔍 View Mask Alone

Alt+Click the mask thumbnail to view the mask itself. Helpful for checking for holes or rough edges in complex masks.

🔗 Unlink Mask from Layer

Click the chain icon between layer and mask thumbnails to move/transform each independently. Useful for repositioning the image inside a mask shape.

CLIPPING MASKS

A Clipping Mask clips one layer's content to the shape of the layer below. Create with Alt+Click between two layers, or Layer → Create Clipping Mask (Ctrl+Alt+G).

Clipping Mask Uses
# Photo inside text
1. Type text in large bold font
2. Place photo above text layer
3. Alt+Click between photo and text ← Creates clipping mask
4. Photo is now visible only inside the text shape

# Color grade only on skin
1. Make selection of skin areas
2. Create Curves Adjustment Layer (auto-creates mask from selection)
3. Adjustments affect only the selected skin area

# Texture inside a shape
1. Draw any shape on a layer
2. Place texture/pattern layer above
3. Ctrl+Alt+G ← Clips texture to shape

LIQUIFY TOOL — FACE & BODY

1
Convert to Smart Object first

Right-click layer → Convert to Smart Object. This makes Liquify a non-destructive Smart Filter you can re-edit.

2
Use Face-Aware Liquify for portraits

Photoshop auto-detects faces. Sliders for Eyes (size, height, width, tilt, distance), Nose, Mouth (smile, width, height), Face Shape.

3
Forward Warp (W) for manual adjustments

Push pixels in any direction. Use a large, soft brush with low pressure (20–40%). Subtle adjustments avoid the "warped" look.

4
Reconstruct to undo mistakes

The Reconstruct brush (R) restores areas back toward the original. Use if any area looks unnatural.

FACEBOOK EVENT BANNER — STEP BY STEP

1
New Document: 1200×628px, 72 PPI, RGB

File → New → Width: 1200px, Height: 628px, Resolution: 72, Color Mode: RGB, Background: White. This is the Facebook event cover size.

2
Place Background Image

File → Place Embedded → choose your image. Scale to fill canvas (hold Shift for proportional). Press Enter. It comes in as a Smart Object.

3
Add Dark Overlay for Text Readability

New layer above photo. Fill with black (Alt+Delete). Change blend mode to Multiply, Opacity 40–60%. This makes text readable over busy photos.

4
Add Gradient for Professionalism

New layer. Gradient Tool (G). Linear gradient from transparent black (left) to transparent (right). Blend mode: Normal, Opacity 70%. Creates depth.

5
Add Event Title (Type Tool — T)

Click canvas, type event name. Choose a bold, readable font. 60–90px for title. White color. Add Drop Shadow (distance 4, size 8, opacity 70%).

6
Add Sub-details: Date, Venue, Time

Separate text layers for each detail. 24–36px. Use a lighter font weight or opacity. Space them with consistent alignment.

7
Add Logo & Branding

File → Place Embedded → your logo PNG (with transparent background). Position top-right or bottom-left. Use White color overlay if the logo looks dark.

8
Add Decorative Elements

Search for ornamental brushes, geometric shapes from Elements (free resources). Add subtle decorative borders, sparkles, or pattern backgrounds.

9
Final Check & Export

Zoom out to 25%. Check text readability, spacing, and overall composition. File → Export → Export As → JPEG, quality 90–100%. Save PSD for future editing.

HOW TO WORK AT SCALE IN PHOTOSHOP

1
Method 1: Reduced Scale (Recommended)

For a 10ft×4ft flex at 150 DPI, that's 18000×7200px — very large. Instead, work at 1:10 scale: 1800×720px at 150 DPI. Design everything at this size, then tell the printer to print at 10ft×4ft.

2
Method 2: Inches at Low DPI

Create document: 120 inches × 48 inches at 25 DPI (for a 10×4ft hoarding). This gives 3000×1200px — manageable and prints at the correct physical size.

3
Method 3: cm at 72 DPI

Convert feet to cm (1ft = 30.48cm). Create in cm at 72 DPI. For 3m×1m banner: 300cm × 100cm at 72 DPI = 8504×2835px. This is standard for many print shops in India/Bangladesh.

FONT USAGE RULES FOR DESIGN

📏 Size Guidelines

Bengali display text at 48–120pt for posters. Body text 10–14pt for print, 16–18px for web. Never below 8pt for print. Bengali characters need slightly more space than Latin — add 5–10% extra line height.

🎨 Mixing Bengali & English

When mixing Bengali and English in one design, pair fonts that have similar x-heights. Kalpurush + Georgia pairing works well. Hind Siliguri + Open Sans for modern digital.

📝 Tracking & Kerning

Bengali text needs looser tracking than Latin. Add +20 to +50 tracking for display text. For flex banners where readability at distance matters, add +30 to +80 tracking.

🖨️ Print Font Safety

For print flex at large sizes, stick to fonts with clear, thick letterforms. Thin decorative fonts become unreadable. Bijoy Baijra and Kalpurush Bold are safest for flex printing.

SMART OBJECTS

Non-destructive editing powerhouse — linked files, smart filters, mockups, and embedding — the professional way to work in Photoshop.

WHY USE SMART OBJECTS?

🔄 Non-destructive Transforms

Scale, rotate, distort a Smart Object multiple times without quality loss. Regular layers lose quality every time.

🎛️ Editable Smart Filters

Filters applied to Smart Objects can be re-edited, hidden, or deleted at any time. They're stored as live, adjustable effects.

🔗 Linked Smart Objects

Link an external file. When the original file updates, all linked instances auto-update. Essential for logo updates across multiple files.

📋 Mockup Templates

Smart Objects power device mockup templates. Double-click → edit the content → save → the mockup updates automatically.

📐 Scaled Quality

Smart Objects preserve the original data. Scale down then scale back up — no quality loss. Critical for images you might resize.

♾️ Multiple Instances

Duplicate a Smart Object (Ctrl+J) — all copies share the same data. Edit one and ALL copies update. Perfect for logos used multiple times.

USING MOCKUP TEMPLATES

1
Open a Mockup PSD

Download free mockup PSDs from Freepik, Mockupworld, or Pixeden. Open the file in Photoshop.

2
Find the Smart Object Layer

In the Layers panel, look for a layer with a Smart Object icon (small page icon in corner of thumbnail). It's usually labeled "YOUR DESIGN HERE" or similar.

3
Double-click to Edit

Double-click the Smart Object thumbnail. A new document opens with the placeholder area.

4
Paste Your Design

Delete the placeholder. Paste or place your banner/poster design to fill the area. Scale to fit. Save (Ctrl+S) and close the Smart Object document.

5
See the Result

Return to the main mockup document. Your design now appears inside the mockup with all the perspective, shadows, and lighting of the original scene.

ACTIONS & AUTOMATION

Record repetitive tasks as Actions and run them on hundreds of files automatically with Batch Processing. Save hours of work every week.

RECORDING AN ACTION

1
Open Actions Panel

Window → Actions (Alt+F9). Click "New Action" (⊕) at the bottom. Name it (e.g., "Resize for Web"). Assign a keyboard shortcut. Click Record.

2
Perform the Steps

Do exactly what you want automated: Image → Image Size, Image → Mode, any adjustments, File → Export. Every action you take is recorded.

3
Stop Recording

Click the Stop button (■) in the Actions panel, or press Escape. Your action is now saved.

4
Test on a Single File

Open a different file. Click your action. Click Play (▶). Watch it execute automatically. Check the result before batch processing.

5
Batch Process a Folder

File → Automate → Batch. Set: Action: your action, Source: Folder (select folder of images), Destination: Folder (choose output folder). Click OK. Watch Photoshop process all files automatically.

USEFUL ACTIONS TO CREATE

📸 "Resize for Social"

Resize to 1080×1080, convert to sRGB, sharpen, export as JPG quality 85. Run on all client photos for social media delivery.

🖨️ "Prepare for Print"

Convert to CMYK, check and set resolution to 300 DPI, add bleed marks, export as TIFF. Standardizes all print files.

✨ "Quick Skin Retouch"

Apply Spot Heal at specific settings, add Frequency Separation layers preset, add basic brightness/contrast. Creates a starting point for portrait retouching.

💧 "Watermark"

Place a logo/watermark layer in the corner at 30% opacity, flatten, export as JPG. Protect all photos before sharing/delivery.

EXPORT & SAVE

The right export format for every purpose — social media, web, print, flex, visiting cards, and archiving your PSD source files.

FILE FORMAT GUIDE

FormatUse CaseSettingsFile Size
PSDSource/master file (always save this)Keep all layers, full qualityLarge (100MB+)
JPG/JPEGPhotos, social media, web bannersQuality 8–10 (80–100%), sRGB for screenSmall-Medium
PNGGraphics with transparency, logosPNG-24 for color, PNG-8 for iconsMedium
TIFFProfessional print, flex, hoardingNo compression or LZW, CMYK, 300 DPIVery Large

WebPModern web use, smaller than JPGFile → Export As → WebP, quality 80Very Small

EXPORT WORKFLOWS

Export Commands Reference
# FOR SOCIAL MEDIA / WEB:
File → Export → Export As
  Format: JPEG, Quality: 80-90
  Color Space: sRGB
  Resize: 1080×1080 or required size

# FOR FLEX / PRINT (to print shop):
Image → Flatten Image   ← Merge all layers first
File → Save As → TIFF
  Compression: None (or LZW)
  Color Space: CMYK (or RGB if shop prefers)
  Include: All layers OFF (flattened)

# FOR VISITING CARD (to press printer):
File → Save As → Photoshop PDF
  PDF Preset: "Press Quality"
  Output: CMYK
  Include Vector Data: YES (for type)
  Security: None

# QUICK EXPORT (batch web export):
File → Export → Quick Export as PNG/JPG
  Configured in: File → Export → Export Preferences

# FOR EMAIL / WHATSAPP (optimized size):
File → Export → Save for Web (Legacy)
  Format: JPEG, Quality 60-70
  Image Size: 1200×628 max
  Optimized: YES
💡
Always Save PSD First: Before exporting to any final format, save your PSD source file. The PSD contains all your layers, adjustments, and text — you'll need it if the client requests changes. Establish this habit: PSD = source, JPG/PDF/TIFF = delivery.
